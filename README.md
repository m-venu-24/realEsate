# 🏠 Real Estate Investment Recommendation Engine

An AI-powered web application that helps users make smarter real estate investment decisions by predicting property prices and recommending suitable investment opportunities based on user preferences and market data.

---

## 📌 Overview

The Real Estate Investment Recommendation Engine is designed to assist buyers and investors in finding profitable real estate properties. The system analyzes various property features such as location, area, number of bedrooms, amenities, and market trends to predict property prices and provide personalized recommendations.

This project combines machine learning with a user-friendly web interface to simplify property investment decisions.

---

## ✨ Features

- 🔍 Search properties based on user preferences
- 💰 Predict property prices using Machine Learning
- 📊 Investment recommendations
- 📈 Data visualization and insights
- 🏡 Property comparison
- 📱 Responsive user interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/m-venu-24/realEsate.git
```

### Navigate to the project

```bash
cd realEsate
```

### Create a virtual environment (Optional)

```bash
python -m venv venv
```

### Activate the environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
